## OpenCMS
---
### OpenCMS is a Legal Case Management System built in Python 3.6. It is intended
to be a hackable / customizable starting point for anyone who wants to build and
run their own bespoke CMS.

#### [A working example](https://open-cms-app.herokuapp.com/)

#### To install locally you will need:
* Python 3.6
* Install dependencies from the requirements.txt file
* If you want this actually cloud-hosted, you'll need an aws or google db.
* Procfile is included for Heroku deployment.
